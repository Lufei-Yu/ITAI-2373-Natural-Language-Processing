This project focuses on building an intelligent text analysis system, NewsBot, designed to understand and analyze linguistic patterns across different news categories. The primary objective is to explore how language usage varies in domains such as Business, Politics, Sports, and Entertainment using Natural Language Processing (NLP) techniques.

The system processes a collection of news articles and performs Part-of-Speech (POS) tagging to extract grammatical structures from the text. By quantifying the distribution of nouns, verbs, adjectives, and numerical expressions, the project identifies distinctive linguistic patterns associated with each category. For example, Business articles often emphasize numerical data, while Sports articles contain more action-oriented verbs.

To support analysis, the project computes statistical summaries and visualizes results using tools like heatmaps, enabling clear comparison of POS tag proportions across categories. These insights help reveal how writing style, tone, and informational focus differ depending on the subject matter.

Overall, this project demonstrates the application of NLP for text classification, pattern recognition, and linguistic analysis, providing a foundation for more advanced systems such as automated news categorization, content recommendation, and intelligent information retrieval.
