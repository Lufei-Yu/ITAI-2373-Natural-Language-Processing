# ITAI-2373-Natural-Language-Processing
This repository contains coursework, projects, and experiments for ITAI-2373: Natural Language Processing. It explores fundamental and applied NLP concepts, including text preprocessing, tokenization, part-of-speech tagging, statistical analysis of language patterns, and machine learning techniques for language understanding.

The repository includes:

 Data preprocessing and cleaning pipelines;
 
 POS tagging and linguistic pattern analysis;
 
 NLP model implementations and evaluations;
 
 Experiments using real-world text datasets;
 
 Jupyter notebooks and project reports.

This work demonstrates practical applications of NLP techniques using Python and modern libraries, with a focus on building a strong foundation for advanced language models and AI systems.
